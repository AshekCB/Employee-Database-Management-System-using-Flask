# Employee-Database-Management-System-using-Flask

Employee Database Management System

Overview
This project is a web-based Employee Database Management System built using Flask, a lightweight Python web framework. 
The system allows for efficient management of employee records including adding, updating, deleting, and viewing detailed employee details.
It features a secure login system, a user-friendly interface, and real-time search and filtering functionalities.

Features
User-Friendly Interface: Easy navigation and usability for all users.
Comprehensive Employee Management: Add, update, delete, and view detailed employee records.

Tech Stack
Backend: Flask
Database: MySQL
Frontend: HTML, CSS
Deployment: Github


Usage

Manage Employees:

Add new employee records via the 'Insert' form.
View employee details by clicking on employee names in the list.
